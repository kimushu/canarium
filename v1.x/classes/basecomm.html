transData

  • transData(txdata?: Buffer, estimator?: function): Promise<Buffer>
    • Defined in base_comm.ts:238

    データの送受信を行う

    Parameters

    • Optional txdata: Buffer

      送信するデータ(制御バイトは自動的にエスケープされる。nullの場合は受信のみ)

    • Optional estimator: function

      受信完了まで繰り返し呼び出される受信処理関数。 引数は受信データ全体と、今回の呼び出しで追加されたデータのオフセット。 省略時は送信のみで完了とする。戻り値の解釈は以下の通り。

                   - number : 指定バイト数を受信して受信完了
             - undefined : 追加データを要求
             - Error : エラー発生時のエラー情報
